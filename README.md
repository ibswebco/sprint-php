### Sprint-PHP

**Sprint-PHP** è la _librearia ufficiale_ PHP per le [API Sprint](https://sprt.ibs.ve.it/docs).

Con Sprint puoi integrare nel tuo sistema gestionale la generazione automatica dei Certificati di origine/Visti.

## Prerequisiti

- PHP 8.1, 8.2, 8.3, or 8.4
- Guzzle HTTP client
